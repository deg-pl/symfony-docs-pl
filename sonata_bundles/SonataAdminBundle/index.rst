Pakiet Admin
============

**SonataAdminBundle jest podzielony na 5 pakietów:**

* SonataAdminBundle: zawiera rdzenne biblioteki i usługi
* `SonataDoctrineORMAdminBundle <https://github.com/sonata-project/SonataDoctrineORMAdminBundle>`_: integruje projekt Doctrine ORM z rdzeniem pakietu admin;
* `SonataDoctrineMongoDBAdminBundle <https://github.com/sonata-project/SonataDoctrineMongoDBAdminBundle>`_: integruje MongoDB z rdzeniem pakietu admin (wczesny etap projektu);
* `SonataDoctrinePhpcrAdminBundle <https://github.com/sonata-project/SonataDoctrinePhpcrAdminBundle>`_: integruje PHPCR z rdzeniem pakietu admin (wczesny etap projektu)
* `SonataPropelAdminBundle <https://github.com/sonata-project/SonataPropelAdminBundle>`_: integruje Propel z rdzeniem pakietu admin (wczesny etap projektu)

Stronę demonstracyjną można znaleźć pod adresem http://demo.sonata-project.org/admin/dashboard
(admin jako użytkownik i hasło).
